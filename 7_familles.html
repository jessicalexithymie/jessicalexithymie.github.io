Contexte:

Projet réalisé au cours de ma première année de BTS.
Le projet vise à créer un site web donnant des informations sur des acteurs importants de 7 grandes familles de l'informatique.
Le but de ce projet était de découvrir l'HTML et la gestion de projet avec git.

Déroulé du projet:

Le projet a été réalisé avec l'outil git afin de partager le code et de chacun pouvoir travailler de notre côté.
Ce projet nous a permis de travailler sur un exemple concret d'usage de HTML et de CSS, et nous a fait rechercher comment réaliser certaines choses avec.
Le but principal de ce projet était pour nous de nous familiariser avec le web statique avant d'aborder des choses plus complexes.
Pour apprendre à utiliser le HTML et le CSS, nous avons pu utiliser les documentations de ces deux langages. Pour travailler, nous avions un cahier des charges à respecter, ainsi qu'une convention de nommage des routes pour qu'une fois les sites des 7 groupes combinés, on puisse se déplacer entre les pages.
